# 🌐 PHP User Management System

A responsive **User Management System** developed using **PHP, MySQL, HTML, CSS, and JavaScript** as part of my Web Development Internship.

---

## 🚀 Features

* 🔐 User Registration & Login
* 🔑 Password Hashing for Security
* 👥 Role-Based Access Control (Admin & User)
* 👤 User Profile Management
* 📷 Profile Picture Upload
* ✏️ Edit User Information
* 🗑️ Delete User
* 🌤️ Weather API Integration
* 📱 Responsive Design
* ✅ Client-side & Server-side Validation
* 🚪 Secure Logout using Sessions

## ⚙️ Installation

1. Install XAMPP.
2. Copy the project folder into:

```
htdocs/
```

3. Start Apache and MySQL.
4. Create a MySQL database.
5. Import the SQL database file.
6. Update database credentials in `db.php`.
7. Open:

```
http://localhost/INTERNSHIP-TASK
```
